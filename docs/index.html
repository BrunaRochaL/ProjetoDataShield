O projeto DataShield visa desenvolver uma extensão para o Google que protege os usuários, detectando sites falsos e e-mails de phishing, fornecendo alertas instantâneos para uma navegação online mais segura. Além disso, o projeto de pesquisa tem como objetivo realizar uma avaliação comparativa entre várias técnicas de algoritmos de classificação e características diversas, a fim de aprimorar a detecção de phishing.
